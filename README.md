# 推箱子游戏

## 项目背景

这是一个基于Vue3的推箱子游戏，旨在提供一个有趣且具有挑战性的游戏体验。

## 项目目标

- 提供多关卡的推箱子游戏体验。
- 支持键盘控制、撤销和重置功能。
- 使用Canvas进行渲染，提供流畅的视觉效果。

## 功能特点

- 多关卡支持：玩家可以挑战不同难度的关卡。
- 键盘控制：使用键盘方向键来控制人物移动。
- 撤销功能：允许玩家撤销上一步操作。
- 重置功能：玩家可以随时重置当前关卡。

## 安装和运行指南

1. 克隆项目到本地：
   ```bash
   git clone <repository-url>
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动项目：
   ```bash
   npm run dev
   ```

## 开发者指南

- 项目目录结构：
  - `src/views/pushBox/`: 包含游戏的主要视图组件。
  - `src/assets/`: 存放游戏相关的图片和样式文件。
- 主要文件：
  - `GameControl.vue`: 控制游戏逻辑的组件。
  - `GameCanvas.vue`: 负责游戏画面渲染的组件。
